# 3.2.2 — See Total Estimated Duration of the Roadtrip

**Epic:** Route Calculation
**Feature:** Trip Totals

## User Story

As a user, I want to see the total estimated duration of the entire roadtrip so that I can understand how long the journey will take.

## Description

The trip overview displays the sum of all leg durations as the total estimated driving time. This is pure driving time — it does not include time spent at stops.

## Acceptance Criteria

- Total duration is displayed on the trip overview screen.
- It equals the sum of all individual leg durations.
- It is clearly labelled as driving time only (does not include stop duration).
- It updates when the route is recalculated.
- If any leg is uncalculated, the total is shown as unavailable or partial with a warning.

## Technical Notes

- Computed from stored `Leg.durationSeconds` values — no additional API call needed.
- Future enhancement: allow users to set a "time at stop" per stop, and show total trip time (driving + stops) alongside pure driving time.
