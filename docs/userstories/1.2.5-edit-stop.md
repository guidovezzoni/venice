# 1.2.5 — Edit a Stop

**Epic:** Trip Management
**Feature:** Stop Management

## User Story

As a user, I want to edit a stop in my roadtrip so that I can correct or change a location.

## Description

The user can select an existing stop and change its location via place search. The stop's name and coordinates are updated. This applies to any stop including starting point and destination.

## Acceptance Criteria

- The user can tap an existing stop to edit it.
- Editing opens the place search to select a new location.
- The stop's name, latitude, and longitude are updated.
- The stop's order remains unchanged.
- Previously calculated legs involving this stop are invalidated.

## Technical Notes

- Invalidation affects at most two legs: the one arriving at this stop and the one departing from it.
