# 1.2.4 — Reorder Stops

**Epic:** Trip Management
**Feature:** Stop Management

## User Story

As a user, I want to reorder stops in my roadtrip so that I can adjust the sequence of my journey.

## Description

The user can change the order of intermediate stops via drag-and-drop or move-up/move-down actions. The starting point and destination remain fixed at the first and last positions respectively.

## Acceptance Criteria

- The user can reorder intermediate stops.
- The starting point (first) and destination (last) cannot be moved.
- The UI provides a clear drag-and-drop or reorder interaction.
- After reordering, previously calculated legs are invalidated.

## Technical Notes

- Reordering updates the `order` field on affected `Stop` records.
- Invalidating legs means clearing cached `Leg` data and showing a prompt to recalculate (see 3.1.1).
- Jetpack Compose has drag-and-drop support via `LazyColumn` reorderable modifiers or third-party libraries.
