# 3.1.2 — See Distance for Each Leg

**Epic:** Route Calculation
**Feature:** Leg Calculation

## User Story

As a user, I want to see the distance for each leg of the trip so that I can understand how far each segment is.

## Description

After route calculation, each leg displays the distance between its two stops. Distance is shown in a human-readable format (km or miles, depending on locale).

## Acceptance Criteria

- Each leg in the trip view shows its distance.
- Distance is displayed in kilometres (with one decimal place) for metric locales or miles for imperial locales.
- If the route has not been calculated yet, distance is shown as unavailable (e.g., "—").

## Technical Notes

- The Google Directions API returns distance in metres (`legs[].distance.value`). Store as integer metres in the database; format for display in the UI layer.
- Locale-based unit selection can use `Locale.getDefault()` to determine metric vs imperial.
