# 1.2.6 — Remove a Stop

**Epic:** Trip Management
**Feature:** Stop Management

## User Story

As a user, I want to remove a stop from my roadtrip so that I can simplify my journey or correct a mistake.

## Description

The user can delete an intermediate stop. The remaining stops reorder to close the gap. Removing the starting point or destination requires special handling — the adjacent stop is promoted.

## Acceptance Criteria

- The user can remove any intermediate stop.
- Remaining stop orders are recalculated to be contiguous.
- If the starting point is removed, the next stop becomes the new starting point.
- If the destination is removed, the previous stop becomes the new destination.
- A trip cannot have fewer than zero stops (removing the last stop leaves an empty trip).
- Previously calculated legs involving the removed stop are invalidated.

## Technical Notes

- Provide a confirmation dialog before deletion to prevent accidental removal.
- Invalidation affects the legs on either side of the removed stop; they merge into a single new uncalculated leg.
