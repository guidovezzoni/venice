# 3.2.1 — See Total Distance of the Roadtrip

**Epic:** Route Calculation
**Feature:** Trip Totals

## User Story

As a user, I want to see the total distance of the entire roadtrip so that I can gauge the overall scale of my journey.

## Description

The trip overview displays the sum of all leg distances as the total trip distance.

## Acceptance Criteria

- Total distance is displayed on the trip overview screen.
- It equals the sum of all individual leg distances.
- It updates when the route is recalculated.
- If any leg is uncalculated (route is stale), the total is shown as unavailable or partial with a warning.

## Technical Notes

- Computed from stored `Leg.distanceMeters` values — no additional API call needed.
- Can be calculated in the ViewModel or as a derived property on the Trip domain model.
- Also shown in the trip list (1.1.2) as a summary.
