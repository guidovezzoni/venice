# 1.1.2 — See a List of My Roadtrips

**Epic:** Trip Management
**Feature:** Trip CRUD

## User Story

As a user, I want to see a list of my roadtrips so that I can select one to view or edit.

## Description

A screen that lists all saved roadtrips. Each entry shows the trip name, number of stops, and total distance/duration if a route has been calculated. The list serves as the app's home screen.

## Acceptance Criteria

- All saved trips are displayed in a scrollable list.
- Each item shows at minimum: trip name, number of stops.
- If route has been calculated, also show total distance and estimated duration.
- Tapping a trip navigates to the trip detail/overview screen.
- Empty state is shown when no trips exist, with a prompt to create one.

## Technical Notes

- **Architecture:** Expose a `Flow<List<Trip>>` from the Repository so the list updates reactively when trips are added or modified.
- This screen will be the entry point of the app, replacing the current `Greeting("Android")` placeholder in `MainActivity`.
